Addressing Modes

Instruction of program has to operate on a data the method of specifying the data to be operated with that instruction is called addressing

different technique of the addressing the data actresses item is addressing modes

1 Immediate

  • The opcode ends with the letter “I” ( referring to the word “Immediate” )
  • 8/16 bit data is in instructions
  • All branch instructions are in immediate addressing mode
  • No need to specify address of operand
  • MVI A,63H        ADI 05H        LXI H,0050H

2 Register

  • Specify source destination address ie both registers  in 8085
  • MOV B,C        
  • ADD D

3 Direct

  • Data is directly copied from the given address to the register.
  • Eg: - LDA 3000H (The content at the location 3000H is copied to the accumulator, A).

4 Indirect

  • The data is transferred from the address pointed by the data in a register pair to the other register.
  • Eg 1: - MOV A, M* (data is transferred from the memory location pointed by the register pair HL to the accumulator. )
  • Eg 2: - LDAX D ( the data at the memory location stored in DE register pair, is loaded in the accumulator)

5 Implied

  • Opcode itself specifies the operand implicitly.
  • CMA         , STC , HLT

*The data, stored at memory location pointed by HL pair, is referred by “M”.
