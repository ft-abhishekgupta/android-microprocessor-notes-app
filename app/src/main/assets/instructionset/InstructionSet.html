Meaning of used Symbols:

A = Accumulator

R = Register

M = Data present at memory address stored in HL Pair

8bit = Data

S = Sign Bit

Z = Zero Bit

AC = Auxilliary Carry

P = Parity Bit

CY = Carry Bit

F = Opcode Fetch = 4 t-states

R = Memory Read = 3 t-states

W = Memory Write = 3 t-states

S = 6 t-states

Addressing modes

Immediate = 8bit data

Register = registers

Direct = 16bit addr

Indirect =RP (register pair)

Implied = No  address mentioned explicitly  (address is referenced implicitly by the opcode itself)

The various addressing modes are discussed in detail later.
