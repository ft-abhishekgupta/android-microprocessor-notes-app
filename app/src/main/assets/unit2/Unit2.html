Instructions

Binary code inside a microprocessor to perform a specific function

INSTRUCTION SET

Entire group of instruction that determine what functions the microprocessor can perform

Instruction word size

1 byte MOV C,A|ADD B|CMA

2 byte MVI A,32H|MVI B,14H

3 byte LDA 2050H|JMP 2085H

There are 5 types of instructions:

  1.  data transfer/control
  2.  arithmetic
  3.  logical
  4.  branching
  5.  machine control

LOOPING

Programming technique used to instruct microprocessor to repeat a task.

Loop is setup by instructing microprocessor to change sequence of execution by repeating a task

Done by jump instruction -        Continuous Loop - counter

                                        Conditional Loop - condition

M needs counter , flags

        Counter is set up by loading count in reg

        Counting by INR/DCR

        Loop by conditional jump statement

        End indicated by flag

COUNTER

Designed by loading number in reg and using INR/DCR

Loop to update and check value

There must be time delay to observe count

INDEXING

is used to specify the elements of an array of numbers , number of elements to be worked on.

Specifying the location of the data is Indexing.

The starting location of the data can be specified by loading the memory ad­dress into a register pair and using the register pair as a memory pointer or index.
