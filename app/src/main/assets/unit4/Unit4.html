Conversions:

BCD-to-Binary conversion

Algorithm:

  1. Separate 8bit packed BCD to 2 4bit unpacked BCD
  2. Convert all digit to its binary
  3. Add both binary numbers

Program:

START        :LXI SP,STACK

        LXI H,INBUF

LXI B,OUTPUT

MOV A,M

CALL BCDBIN

STAX B

HLT

BCDBIN:PUSH B

        PUSH D

        MOV B,A

        ANI 0FH

        MOV C,A

        MOV A,B

        ANI F0H

        JZ BCD1

        RRC

        RRC

        RRC

        RRC

        MOV D,A

XRA A

MVI E,0AH

SUM:        ADD E

        DCR D

        JNZ SUM

BCD1: ADD C

        POP C

        POP B

        RET

Binary-to-BCD conversion

Algorithm

  1. If number less than 100 goto step 2 else / by 100 again n again , quotient is BCD3
  2. If number less than 10 goto step 3 else / by 10 again n again , quotient is BCD2
  3. Remainder is BCD1

Program

START:LXI SP,STACK

        LXI H,BINBYT

        MOV A,M

        CALL PWRTEN

        HLT

PWRTEN:LXI H,OUTBUF

        MVI B,64H

        CALL BINBCD

        MVI B,0AH

        CALL BINBCD

        MOV M,A

        RET

BINBCD:MVI M,FFH

NXTBUF:INR M

        SUB B

        JNC NXTBUF

        ADD B        

        INX H

        RET

Binary-to-ASCII

Algorithm

  1. Transfer byte in A
  2. Separate nibble
  3. Call
  4. Store each n XX60,XX61

Program

LXI SP,STACK

LXI H,XX50

LXI D,XX60

MOV A,M

MOV B,A

RRC

RRC

RRC

RRC

CALL ASCII

STAX D        1st ascii

INX D

MOV A,B        2nd digit

CALL ASCII

STAX D        2nd ascii

HLT

ASCII:ANI 0FH          //(mask higher bit)

        CPI 0AH

        JC CODE

        ADI 07H        add 07 to code A-F

CODE:ADI 30H        for 0-9

        RET

ASCII-to-Binary code conversion

ASCBIN:SUI 30H

        CPI 0AH

        RC

        SUI 07H

        RET
